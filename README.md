# Smart Home IDS Project

## Overview

The Smart Home IDS Project captures network traffic, analyzes the captured data, and displays the results through a web-based dashboard.

> **Note:** For testing purposes, use an Ubuntu Desktop virtual machine.

---

## Setup

### 1. Create the Project Directory

Create a new directory for the project and navigate into it:

```bash
mkdir SeniorProject-SmartHomeIDS
cd SeniorProject-SmartHomeIDS
```

### 2. Add the Project Files

Copy all project source files into the `SeniorProject-SmartHomeIDS` directory.

---

## Capturing Network Traffic

Run the following command to capture network traffic and save it to a PCAP file:

```bash
sudo tcpdump -w capture.pcap
```

This command creates a file named `capture.pcap` containing the network traffic captured on the VM.

When you have collected enough traffic, stop the capture by pressing:

```text
Ctrl + C
```

---

## Analyzing the PCAP File

After creating the PCAP file, run the analyzer script:

```bash
python3 pcap_analyzer.py
```

If the script is named `pcap.analyzer`, use:

```bash
python3 pcap.analyzer
```

The script will analyze the captured traffic and generate a JSON file in the project directory.

---

## Starting the Web Server

From the same project directory, start a Python HTTP server:

```bash
python3 -m http.server 8000
```

---

## Viewing the Dashboard

Find the IP address of your Ubuntu VM:

```bash
hostname -I
```

Open a web browser in the VM and navigate to:

```text
http://<VM_IP_ADDRESS>:8000
```

Example:

```text
http://192.168.1.100:8000
```

The dashboard should load and display the analyzed network traffic data.

---

## Workflow Summary

1. Create the project directory.
2. Copy the project files into the directory.
3. Capture network traffic:

   ```bash
   sudo tcpdump -w capture.pcap
   ```

4. Stop the capture with **Ctrl + C**.

   ```bash

   ```

   ```bash

   ```

7. Open a browser and navigate to:

   ```text

   ```

8. Review the IDS dashboard and analysis results.

## Troubleshooting

### Permission Denied When Running tcpdump

Use:

```bash
sudo tcpdump -w capture.pcap
```

### Cannot Access the Dashboard

Verify that:

- The HTTP server is running.
- You are using the correct IP address.
- Port `8000` is not blocked by a firewall.

### Python Not Installed

Check your Python installation:

```bash
python3 --version
```

Install Python if necessary:

```bash
sudo apt update
sudo apt install python3
```
