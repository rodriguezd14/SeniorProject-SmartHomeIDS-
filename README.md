Smart Home IDS Project
Overview

The Smart Home IDS Project is a simple intrusion detection system workflow that captures network traffic, analyzes it, and presents the results through a web interface.

Note: For testing purposes, use an Ubuntu Desktop virtual machine.

Prerequisites
Ubuntu Desktop VM
Python 3 installed
tcpdump installed
Web browser

Install tcpdump if needed:

sudo apt update
sudo apt install tcpdump
Setup
1. Create the Project Directory

Create a new directory for the project:

mkdir SeniorProject-SmartHomeIDS
cd SeniorProject-SmartHomeIDS
2. Add the Project Files

Copy and paste all project source files into the SeniorProject-SmartHomeIDS directory.

Capturing Network Traffic

Run the following command to capture network traffic and save it to a PCAP file:

sudo tcpdump -w capture.pcap

This command creates a file named capture.pcap containing the captured network traffic.

Press Ctrl + C when you have collected enough traffic.

Analyzing the PCAP File

After creating the PCAP file, run the analyzer script:

python3 pcap_analyzer.py

If your file is actually named pcap.analyzer, use:

python3 pcap.analyzer

The script will analyze the captured traffic and generate a JSON file containing the results. The JSON file will be saved in the project directory.

Starting the Web Server

From the same project directory, start a Python HTTP server:

python3 -m http.server 8000

You should see output similar to:

Serving HTTP on 0.0.0.0 port 8000
Viewing the Dashboard

Determine the IP address of your Ubuntu VM:

ip addr

or

hostname -I

Open a web browser inside the VM and navigate to:

http://<VM_IP_ADDRESS>:8000

Example:

http://192.168.1.100:8000

The dashboard should load and display the analyzed network traffic data.

Troubleshooting
Permission Denied When Running tcpdump

Run tcpdump with elevated privileges:

sudo tcpdump -w capture.pcap
Cannot Access the Dashboard

Verify that:

The HTTP server is running.
You are using the correct VM IP address.
Port 8000 is not blocked by a firewall.
Python Command Not Found

Verify Python is installed:

python3 --version

If necessary:

sudo apt install python3
